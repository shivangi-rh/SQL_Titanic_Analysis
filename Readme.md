# Titanic Analysis using SQL
## Objective
    The objective of this project is to analyze titanic dataset and understand the surivial rates based on passenger's class,age,sex,fare.
## Dataset
    Kaggle Titanic Passenger Dataset
## Key Analysis Performed
    -Exploring Data using SELECT
    -FIltering Data using WHERE clause
    -Grouping the Data by GROUP BY clause
    -Survival Analysis by pclass,fare,sex
    -Aggregate functions(max,min,count,sum,avg)
## Key Insights
    -Female passengers had higher survival rate than male passengers.
    -Passengers of 1st class were more likely to survive compared to others.
    -Passengers of higher class paid higher fees and had better survival chance.
